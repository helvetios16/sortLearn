# Gemini Project Instructions

Este archivo contiene convenciones y preferencias para los agentes de IA que trabajen en este repositorio.

## Estructura de Commit

Cuando se solicite un commit, se debe seguir estrictamente la siguiente estructura:

```text
 <tipo>(<alcance>): <resumen corto>

    - <Detalle del cambio 1>
    - <Detalle del cambio 2>
    - <Detalle del cambio 3>
```

### Ejemplo:
 refactor(visualize): update UI and logic for bottle visualization and sorting

    - Refactor Bottle.vue, Visualize.vue, and VisualizeNew.vue with formatting fixes and UI improvements.
    - Update scale logic and drag-and-drop interactions.
    - Enhance visual feedback and animations for sorting steps.
